## 用法

```sh
$ python run.py -h
usage: run.py [-h] [-b]

Create Hidden Markov Model(HMM) for POS Tagging

optional arguments:
  -h, --help  show this help message and exit
  -b          use big data
  --file FILE, -f FILE  set where to store the model
# e.g. 
$ python run.py -b
```
